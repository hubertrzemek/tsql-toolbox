# Keyboard Shortcuts in SQL Server Management Studio (SSMS)

| Shortcut                 | Description                        |
| ------------------------ | ---------------------------------- |
| **F5 / Ctrl + E**        | Execute selected query             |
| **Ctrl + Shift + E**     | Execute with actual execution plan |
| **Ctrl + K, Ctrl + C**   | Comment selected lines             |
| **Ctrl + K, Ctrl + U**   | Uncomment selected lines           |
| **Ctrl + Shift + U**     | Convert selected text to UPPERCASE |
| **Ctrl + Shift + L**     | Convert selected text to lowercase |
| **Alt + Shift + Arrows** | Column editing                     |
| **Ctrl + H**             | Find and replace                   |
| **Ctrl + L**             | Display estimated execution plan   |
| **Ctrl + R**             | Show / Hide results pane           |
| **Ctrl + Tab**           | Switch between query windows       |
| **Ctrl + G**             | Go to specific line number         |
| **F8**                   | Open Object Explorer               |
| **Ctrl + T**             | Results to text                    |
| **Ctrl + D**             | Results to grid                    |
| **Ctrl + Shift + F**     | Results to file                    |
| **Ctrl + N**             | New query                          |
